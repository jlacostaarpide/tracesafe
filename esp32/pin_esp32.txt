⚠️  PINES PROHIBIDOS EN ESTE CHIP:
    GPIO  6-11 → flash interna del chip (usar los destruye)
    GPIO  34,35,36,39 → solo entrada, no pueden dar salida
    GPIO  1, 3 → UART TX/RX (Serial Monitor), evitar

  TODOS los cables cambian de sitio porque los pines del
  Arduino Uno colisionan con la flash interna del ESP32.

  Arduino Uno → ESP32 DEVKITV1:
    RFID SCK  : 13  →  18
    RFID MOSI : 11  →  23
    RFID MISO : 12  →  19
    RFID SS   : 10  →   5
    RFID RST  :  9  →  17
    Servo     :  3  →   4
    LED rojo  :  8  →  25
    LED naranj:  7  →  26
    LED verde :  6  →  27
    LCD RS    :  A1 →  14
    LCD EN    :  A2 →  22
    LCD D4    :  A3 →  16
    LCD D5    :  A4 →  21
    LCD D6    :  A5 →  32
    LCD D7    :   5 →  33
    Botón     :   2 →   2  (este sí se queda igual)

  TABLA COMPLETA DE CONEXIONES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ┌─────────────────────────────────────────────────────────────┐
  │  MÓDULO RFID — MFRC522                                      │
  ├──────────────┬──────────────┬───────────────────────────────┤
  │  Pin MFRC522 │  ESP32       │  Notas                        │
  ├──────────────┼──────────────┼───────────────────────────────┤
  │  VCC         │  3.3V        │  ⚠️ Solo 3.3V, nunca 5V       │
  │  GND         │  GND         │                               │
  │  SCK         │  GPIO 18     │  VSPI_CLK                     │
  │  MOSI        │  GPIO 23     │  VSPI_MOSI                    │
  │  MISO        │  GPIO 19     │  VSPI_MISO                    │
  │  SDA (SS)    │  GPIO  5     │  Chip Select                  │
  │  RST         │  GPIO 17     │                               │
  └──────────────┴──────────────┴───────────────────────────────┘

  ┌─────────────────────────────────────────────────────────────┐
  │  SERVO MOTOR (SG90 / MG90S)                                 │
  ├──────────────┬──────────────┬───────────────────────────────┤
  │  Cable       │  Conectar a  │  Notas                        │
  ├──────────────┼──────────────┼───────────────────────────────┤
  │  Señal (naranja/amarillo) │ GPIO 4 │ PWM                   │
  │  VCC (rojo)  │  5V (VIN)    │  Usar VIN del DEVKITV1        │
  │  GND (marrón)│  GND         │  GND común con ESP32          │
  └──────────────┴──────────────┴───────────────────────────────┘

  ⚠️ Si el servo provoca reinicios, aliméntalo con fuente
     externa de 5V con GND común al ESP32.

  ┌─────────────────────────────────────────────────────────────┐
  │  LEDs (con resistencia de 220 Ω en serie cada uno)          │
  ├──────────────┬──────────────┬───────────────────────────────┤
  │  LED         │  ESP32       │                               │
  ├──────────────┼──────────────┼───────────────────────────────┤
  │  Rojo        │  GPIO 25     │                               │
  │  Naranja     │  GPIO 26     │                               │
  │  Verde       │  GPIO 27     │                               │
  └──────────────┴──────────────┴───────────────────────────────┘

  Esquema por LED:
    GPIO → [220 Ω] → Ánodo (+, pata larga) → Cátodo (-, corta) → GND

  ┌─────────────────────────────────────────────────────────────┐
  │  BOTÓN DE EMERGENCIA                                        │
  ├──────────────┬──────────────┬───────────────────────────────┤
  │  Pin botón   │  Conectar a  │  Notas                        │
  ├──────────────┼──────────────┼───────────────────────────────┤
  │  Un extremo  │  GPIO 13     │                               │
  │  Otro extremo│  GND         │  Sin resistencia (usa PULLUP) │
  └──────────────┴──────────────┴───────────────────────────────┘

  ┌─────────────────────────────────────────────────────────────┐
  │  LCD 16x2 — MODO 4 BITS (sin módulo I2C)                    │
  ├──────────────┬──────────────┬───────────────────────────────┤
  │  Pin LCD     │  Conectar a  │  Notas                        │
  ├──────────────┼──────────────┼───────────────────────────────┤
  │  1  VSS      │  GND         │                               │
  │  2  VDD      │  5V (VIN)    │                               │
  │  3  V0       │  Potenciómetro 10kΩ │ Ajusta el contraste   │
  │  4  RS       │  GPIO 14     │                               │
  │  5  RW       │  GND         │  Siempre a GND                │
  │  6  E (EN)   │  GPIO 22     │                               │
  │  7–10 D0–D3  │  Sin conectar│  Modo 4 bits                  │
  │  11 D4       │  GPIO 16     │                               │
  │  12 D5       │  GPIO 21     │                               │
  │  13 D6       │  GPIO 32     │                               │
  │  14 D7       │  GPIO 33     │                               │
  │  15 A (+BL)  │  5V + 220Ω  │  Retroiluminación             │
  │  16 K (-BL)  │  GND         │  Retroiluminación             │
  └──────────────┴──────────────┴───────────────────────────────┘

  Potenciómetro de contraste (V0):
    Extremo 1 → 5V  |  Centro → Pin 3 LCD  |  Extremo 3 → GND

  RESUMEN DE TODOS LOS GPIO USADOS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  GPIO 13  →  Botón emergencia
  GPIO  4  →  Servo (señal PWM)
  GPIO  5  →  RFID SS (SDA)
  GPIO 14  →  LCD RS
  GPIO 16  →  LCD D4
  GPIO 17  →  RFID RST
  GPIO 18  →  RFID SCK
  GPIO 19  →  RFID MISO
  GPIO 21  →  LCD D5
  GPIO 22  →  LCD EN
  GPIO 23  →  RFID MOSI
  GPIO 25  →  LED Rojo
  GPIO 26  →  LED Naranja
  GPIO 27  →  LED Verde
  GPIO 32  →  LCD D6
  GPIO 33  →  LCD D7

  ESP32 DEVKITV1  →  USB al ordenador
  RFID            →  3.3V del ESP32 (pin 3V3)
  LCD             →  5V del ESP32 (pin VIN)
  Servo           →  5V del ESP32 (pin VIN)
  LEDs            →  GPIO directo con resistencia 220Ω
  GND             →  Todos al mismo GND
